◆ meetingId

std::string NEMeetingParams::meetingId

指定要创建或加入的目标会议ID

  • 加入会议时，该字段必须是一个当前正在进行中的会议ID，不能为空
  • 创建会议时，该字段可使用通过NEAccountService#getPersonalMeetingId返回的个人会议号，或者不指定(置空)。 当不指定会议ID创建会议时，由服务端随机分配一个会议ID
